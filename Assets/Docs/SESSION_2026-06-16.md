# Session Summary — 16 มิถุนายน 2026

## สิ่งที่ทำวันนี้

### 1. วิเคราะห์ Gap vs Frostpunk

สรุปสิ่งที่ Nuclear Re:Mind ขาดเมื่อเทียบกับ Frostpunk แบ่งเป็น 3 ระดับ:

**Critical (ทำให้ไม่รู้สึกเหมือน Frostpunk เลย):**
- ไม่มี Day counter / Deadline countdown
- ไม่มี Speed controls (Pause / 1× / 2×)
- Worker ไม่ได้ assign ต่ออาคาร (แค่ global pool)
- อาคารไม่กินทรัพยากร (มีแค่ production ฝั่งเดียว)
- ไม่มี Alert / Notification system

**Important:** Resource flow breakdown, Exploration system, Buildings หยุดเมื่อถอน worker, End-game summary

**Nice to have:** Law/Policy persistent, Population growth, Radiation zone visual, Quiz per phase

### 2. ระบบทุบอาคาร (Demolition System) ✅

#### ไฟล์ใหม่
| ไฟล์ | หน้าที่ |
|---|---|
| `Assets/Scripts/Systems/DemolitionController.cs` | Singleton จัดการ demolish mode |
| `Assets/Editor/DemolitionSetup.cs` | Menu: NuclearReMind/Setup Demolition System |

#### ไฟล์แก้ไข
| ไฟล์ | สิ่งที่เปลี่ยน |
|---|---|
| `EventManager.cs` | เพิ่ม `OnDemolishModeToggled` event + `RaiseDemolishModeToggled(bool)` |
| `GridManager.cs` | **แก้ bug** — เพิ่ม `_buildingFootprints` dict, subscribe `OnBuildingPlaced`/`OnBuildingRemoved` เพื่อ track + free cells เมื่อทุบ (bug เดิม: cancel construction ก็ไม่ปลด cell) |
| `BuildingSelectionUI.cs` | เพิ่มปุ่ม 🔨 ทุบอาคาร ท้ายแถว hotbar (toggle สีแดง/แดงเข้ม) |
| `PlacementController.cs` | subscribe `OnDemolishModeToggled` → cancel placement เมื่อ demolish เปิด; `BeginPlacement()` ปิด demolish อัตโนมัติ |

#### Logic สำคัญ
- ทุบเสร็จ: **คืน workers เต็ม, energy ที่ใช้ก่อสร้างไม่คืน** (ต่างจาก cancel construction ที่คืนทั้งคู่)
- อาคารที่กำลังก่อสร้าง → block ทุบ, log แจ้งให้ใช้ Cancel ใน BuildingQueueUI
- Cascade เมื่อ `RaiseBuildingRemoved`:
  - `GridManager` → free footprint cells
  - `BuildingVisualSpawner` → destroy sprite
  - `BuildingRegistry` → remove จาก dict
  - `ConstructionController` → no-op (ไม่อยู่ใน queue)

#### วิธีใช้ใน Unity Editor (ต้องทำก่อนเล่น)
1. เปิด Gamescene
2. Menu: `NuclearReMind → Setup Demolition System`

4. กด Play → กดปุ่ม 🔨 ในแถบล่าง
