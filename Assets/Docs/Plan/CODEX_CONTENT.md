### AG-01 · โคบอลต์-60 กับการถนอมอาหาร
```
id: agri_cobalt60
title: โคบอลต์-60 กับการถนอมอาหาร
unlockEvent: ON_CRISIS_FOOD_TRIGGERED
branch: Agriculture

content:
โคบอลต์-60 (Co-60) คือไอโซโทปกัมมันตรังสีที่ปล่อยรังสีแกมมาพลังงานสูง
เมื่อฉายรังสีแกมมาจาก Co-60 ผ่านอาหาร รังสีจะทำลาย DNA ของจุลินทรีย์
แบคทีเรีย เชื้อรา และแมลงศัตรูพืช ทำให้ไม่สามารถเจริญพันธุ์ได้

ข้อสำคัญ: อาหารที่ผ่านการฉายรังสีไม่ได้กลายเป็นกัมมันตรังสี — รังสีแกมมา
ทะลุผ่านอาหารโดยไม่ทิ้งสารตกค้าง เทคนิคนี้ใช้กันทั่วโลก เช่น การฉายรังสี
สตรอว์เบอร์รี มะม่วง และเครื่องเทศ เพื่อยืดอายุการเก็บรักษา

ในเมือง Veltara: เครื่องฉายโคบอลต์-60 ที่สร้างจาก CORE TOWER
ช่วยหยุดการเน่าเสียของเสบียงทั้งคลัง — อัตราการเน่าลดเหลือ 0%

nuclearKnowledge: Co-60 มีครึ่งชีวิต 5.27 ปี ปล่อยรังสีแกมมา 2 เส้น
ที่ 1.17 MeV และ 1.33 MeV — พลังงานสูงพอทะลุกล่องบรรจุภัณฑ์หนาได้
```

### AG-02 · ฉายรังสีแกมมา ทำลายจุลินทรีย์
```
id: agri_irradiation
title: การฉายรังสีถนอมอาหาร (Food Irradiation)
unlockEvent: ON_CRISIS_FOOD_RESOLVED_B
branch: Agriculture

content:
Food Irradiation คือกระบวนการฉายรังสีในปริมาณที่ควบคุมเพื่อลด
จุลินทรีย์ก่อโรคในอาหาร แบ่งเป็น 3 ระดับ:

• Radurization (ปริมาณต่ำ 0.1–1 kGy): ยืดอายุผลไม้ ลดแมลง
• Radicidation (ปริมาณกลาง 1–10 kGy): ทำลาย Salmonella, E.coli
• Radappertization (ปริมาณสูง 10–50 kGy): ฆ่าเชื้อทุกชนิด
  เหมือนการฆ่าเชื้อในกระป๋อง

องค์การอาหารและยาสหรัฐ (FDA) และ WHO รับรองความปลอดภัยของ
อาหารฉายรังสีตั้งแต่ปี 1980 — ปัจจุบันใช้ใน 60 กว่าประเทศ

nuclearKnowledge: หน่วย Gray (Gy) = พลังงาน 1 จูล ที่ดูดซึมต่อสาร 1 กก.
อาหารฉายรังสีใช้ 0.1–10 kGy ซึ่งน้อยกว่าพลังงานที่จะทำให้อาหารร้อนได้
```

### AG-03 · การปรับปรุงพันธุ์พืชด้วยรังสี (Mutation Breeding)
```
id: agri_mutation_breeding
title: การปรับปรุงพันธุ์พืชด้วยรังสี
unlockEvent: ON_CRISIS_FOOD_RESOLVED_A
branch: Agriculture

content:
Mutation Breeding คือการใช้รังสีแกมมาหรือนิวตรอนกระตุ้นให้ DNA พืช
เกิดการกลายพันธุ์แบบสุ่ม จากนั้นคัดเลือกสายพันธุ์ที่มีลักษณะดีขึ้น
เช่น ทนแล้ง ทนโรค หรือให้ผลผลิตสูง

ตัวอย่างในประเทศไทย:
• ข้าว กข6 — พันธุ์ข้าวเหนียว ต้านทานโรคไหม้ ได้จากการฉายรังสีแกมมา
• ข้าว กข10, กข15 — ทนทานต่อสภาพอากาศและโรคแมลง
• ถั่วเหลืองดอยคำ — ผลผลิตสูงขึ้น 30% หลังการปรับปรุงพันธุ์ด้วยรังสี

องค์การอาหารและเกษตรแห่งสหประชาชาติ (FAO) บันทึกพันธุ์พืชที่ได้จาก
Mutation Breeding มากกว่า 3,200 สายพันธุ์ใน 70 ประเทศ

nuclearKnowledge: รังสีแกมมาทำให้เกิด DNA double-strand break
กระตุ้น repair mechanism ของเซลล์ บางครั้งเกิดการกลายพันธุ์ที่มีประโยชน์
อัตราการกลายพันธุ์ที่มีประโยชน์อยู่ที่ประมาณ 1 ใน 1,000
```

### AG-04 · รังสีกับการกลายพันธุ์เชิงบวก
```
id: agri_positive_mutation
title: การกลายพันธุ์เชิงบวก (Positive Mutation)
unlockEvent: ON_AGRIDOME_BUILT
branch: Agriculture

content:
ไม่ใช่การกลายพันธุ์ทุกอย่างที่เป็นอันตราย — ในพืช การกลายพันธุ์จาก
รังสีสามารถสร้างลักษณะที่เป็นประโยชน์ได้:

• ยีนต้านทานโรค: พืชสังเคราะห์สารยับยั้งเชื้อราได้มากขึ้น
• โครงสร้างใหม่: เมล็ดใหญ่ขึ้น ลำต้นแข็งแรงขึ้น
• วงจรชีวิตสั้นลง: เก็บเกี่ยวได้เร็วขึ้น 20–30%

กระบวนการ:
1. ฉายรังสีเมล็ดพันธุ์ด้วย Co-60 หรือ gamma chamber
2. ปลูกรุ่น M1 — คัดทิ้งสายพันธุ์ที่ไม่ดี
3. ปลูกรุ่น M2–M5 — คัดเลือกลักษณะที่ต้องการ
4. ทดสอบภาคสนาม 2–3 ปีก่อนปล่อยใช้จริง

nuclearKnowledge: ปริมาณรังสีที่ใช้ใน mutation breeding อยู่ที่ 100–400 Gy
สำหรับรังสีแกมมา (สูงกว่าการฉายอาหาร 10–40 เท่า) เพราะต้องการให้
เกิดการกลายพันธุ์ ไม่ใช่แค่ทำลายจุลินทรีย์
```

### MED-01 · PET Scan — ตรวจโดยไม่ต้องผ่าตัด
```
id: med_pet_scan
title: PET Scan — มองเห็นความเจ็บป่วยระดับโมเลกุล
unlockEvent: ON_MEDCENTER_BUILT
branch: Medical

content:
PET (Positron Emission Tomography) คือการสแกนที่ฉีดสารเภสัชรังสี
เข้าร่างกาย สารนี้จะสะสมในเนื้อเยื่อที่ใช้พลังงานสูง (เช่น เซลล์มะเร็ง
ที่เจริญเติบโตเร็ว)

กระบวนการ:
1. ฉีด FDG (Fluorodeoxyglucose) ที่ติดฉลากด้วย F-18
2. เซลล์มะเร็งกินกลูโคส → สะสม F-18 ไว้มาก
3. F-18 สลายตัว ปล่อย positron → ชนกับ electron → ปล่อยรังสีแกมมา 2 เส้น
4. กล้องวง 360° จับรังสีทั้ง 2 เส้น → สร้างภาพ 3 มิติ

ข้อดี: เห็นความผิดปกติระดับโมเลกุลก่อนจะมีอาการ
ตรวจพบมะเร็งระยะแรกได้แม่นยำกว่า X-ray ทั่วไปมาก

nuclearKnowledge: F-18 มีครึ่งชีวิต 110 นาที — สั้นพอที่รังสีจะหายไปจาก
ร่างกายภายใน 24 ชั่วโมง แต่ยาวพอให้ทำการสแกนได้ทัน
```

### MED-02 · SPECT — สแกนการทำงานของอวัยวะ
```
id: med_spect_scan
title: SPECT — ติดตามการทำงานของอวัยวะ
unlockEvent: ON_CRISIS_OUTBREAK_TRIGGERED
branch: Medical

content:
SPECT (Single Photon Emission CT) คล้าย PET แต่ใช้ไอโซโทปที่ปล่อย
รังสีแกมมาเพียงทิศเดียว ใช้ Technetium-99m (Tc-99m) เป็นหลัก

การประยุกต์ใช้:
• ตรวจการไหลเวียนเลือดในสมอง (brain perfusion)
• ตรวจการทำงานของหัวใจ (myocardial perfusion)
• ตรวจการกระจายของมะเร็งไปกระดูก (bone scan)
• ตรวจต่อมไทรอยด์และต่อมน้ำเหลือง

ในวิกฤต Malignant Outbreak: SPECT ช่วยระบุว่าคนงานคนไหนมีเซลล์
กลายพันธุ์ลุกลาม เพื่อจัดลำดับการรักษาก่อน-หลัง

nuclearKnowledge: Tc-99m มีครึ่งชีวิต 6.02 ชั่วโมง — เหมาะที่สุดสำหรับ
การวินิจฉัย เพราะสั้นพอให้รังสีหมดไปในวัน แต่ยาวพอให้ทีมแพทย์ทำงานได้
ผลิตจากการสลายตัวของ Molybdenum-99 ในเครื่อง generator
```

### MED-03 · Targeted Radionuclide Therapy
```
id: med_radionuclide_therapy
title: การรักษาด้วยสารรังสีเฉพาะจุด
unlockEvent: ON_CRISIS_OUTBREAK_RESOLVED_B
branch: Medical

content:
Targeted Radionuclide Therapy (TRT) คือการฉีดสารรังสีที่ออกแบบให้
จับกับเซลล์มะเร็งโดยเฉพาะ แล้วทำลายเซลล์เหล่านั้นจากภายใน

หลักการ:
• สร้าง "ขีปนาวุธนำวิถี" — เชื่อม Antibody กับ Radionuclide
• Antibody นำทางไปหา receptor บนเซลล์มะเร็ง
• Radionuclide ปล่อยรังสี beta หรือ alpha — ทำลาย DNA เซลล์มะเร็ง
• เซลล์ข้างเคียงได้รับรังสีน้อยกว่า Chemo หรือ Radiation Therapy ทั่วไป

ตัวอย่างที่ใช้จริง:
• Lutetium-177 + PSMA → รักษามะเร็งต่อมลูกหมาก
• Iodine-131 → รักษามะเร็งต่อมไทรอยด์ (ต่อมไทรอยด์ดูดซึม I-131 โดยธรรมชาติ)
• Yttrium-90 → รักษามะเร็งตับ

nuclearKnowledge: รังสี beta มีระยะทะลุในเนื้อเยื่อสั้นมาก (< 2 มม.)
จึงทำลายเฉพาะเซลล์มะเร็งในรัศมีใกล้ ๆ ลดผลข้างเคียงกับเซลล์ปกติ
```

### MED-04 · ปริมาณรังสีที่ปลอดภัย
```
id: med_radiation_dose
title: ปริมาณรังสีและขีดจำกัดความปลอดภัย
unlockEvent: ON_ZONE_A_OPENED
branch: Medical

content:
ปริมาณรังสีวัดด้วยหน่วย Sievert (Sv) — คำนึงถึงทั้งพลังงานรังสีและ
ความเป็นอันตรายต่อเนื้อเยื่อ

ระดับอ้างอิง:
• 2.4 mSv/ปี — รังสีพื้นหลังโดยเฉลี่ย (ธรรมชาติ)
• 20 mSv/ปี — ขีดจำกัดสำหรับนักรังสีวิทยา (ICRP)
• 100 mSv — เริ่มเพิ่มความเสี่ยงมะเร็งวัดได้ (ตลอดชีวิต)
• 1,000 mSv (1 Sv) — เจ็บป่วยจากรังสีเฉียบพลัน (Acute Radiation Syndrome)
• 4,000–5,000 mSv — อัตราการเสียชีวิต 50% (LD50)

หลัก ALARA (As Low As Reasonably Achievable):
ลดการรับรังสีให้ต่ำที่สุดเท่าที่ทำได้ในทางปฏิบัติ
→ เป็นหลักจริยธรรมพื้นฐานของการป้องกันรังสีสากล

nuclearKnowledge: รังสีแกมมา 1 Gy = 1 Sv (weighting factor = 1)
แต่รังสีอัลฟา 1 Gy = 20 Sv (weighting factor = 20) เพราะทำลายเนื้อเยื่อ
ได้มากกว่าต่อพลังงานเท่ากัน
```

### MED-05 · การกลายพันธุ์ของเซลล์จากรังสี
```
id: med_cell_mutation
title: รังสีไอออไนซ์กับ DNA
unlockEvent: ON_CRISIS_OUTBREAK_TRIGGERED
branch: Medical

content:
รังสีไอออไนซ์มีพลังงานสูงพอที่จะดึงอิเล็กตรอนออกจากอะตอมในเซลล์
ทำให้เกิดอนุมูลอิสระที่ทำลาย DNA

ประเภทของความเสียหาย:
• Single-strand break: เซลล์ซ่อมแซมได้เองส่วนใหญ่
• Double-strand break: อันตรายที่สุด ซ่อมยาก
  → อาจนำไปสู่การตายของเซลล์ หรือการกลายพันธุ์ที่ทำให้เป็นมะเร็ง

ผลตามระยะเวลา:
• ผลฉับพลัน (< 1 เดือน): คลื่นไส้ ผมร่วง ระบบภูมิคุ้มกันพัง
• ผลระยะยาว (ปี–ทศวรรษ): ความเสี่ยงมะเร็งสูงขึ้น โดยเฉพาะมะเร็งเม็ดเลือด
• ผลต่อทายาท: การกลายพันธุ์ในเซลล์สืบพันธุ์ (Heritable effects)

กลุ่มเสี่ยงสูงที่สุด: เด็กและทารกในครรภ์ เพราะเซลล์แบ่งตัวเร็ว
ความเสียหายจึงกระจายและขยายผลได้มากกว่า

nuclearKnowledge: นี่คือเหตุผลที่หลัก ALARA ให้ความสำคัญกับการปกป้อง
กลุ่มเปราะบาง — เด็ก ผู้ป่วย และหญิงมีครรภ์ต้องได้รับรังสีน้อยที่สุด
```

### ENV-01 · ALARA Principle
```
id: env_alara
title: หลัก ALARA — ความรับผิดชอบต่อรังสี
unlockEvent: ON_EMERGENCY_DECREE_AVAILABLE
branch: Environment

content:
ALARA (As Low As Reasonably Achievable) คือหลักการระหว่างประเทศ
สำหรับการป้องกันรังสี กำหนดโดย ICRP (International Commission on
Radiological Protection)

3 วิธีลดการรับรังสี:
1. เวลา (Time): ลดเวลาที่อยู่ใกล้แหล่งรังสี
2. ระยะทาง (Distance): ห่างออก 2 เท่า รังสีลด 4 เท่า (กฎกำลังสอง)
3. การป้องกัน (Shielding): ตะกั่ว คอนกรีต หรือน้ำ ดูดซับรังสีแกมมา

ผู้ที่ต้องได้รับการปกป้องเป็นพิเศษ:
• เด็ก (เซลล์แบ่งตัวเร็ว — ไวรังสีมากกว่า)
• ผู้ป่วย (ระบบซ่อมแซม DNA อ่อนแอกว่า)
• หญิงมีครรภ์ (ทารกในครรภ์ไวรังสีสูงมาก)

ในวิกฤต Veltara: การบังคับผู้ป่วยหรือเด็กเข้าพื้นที่รังสีสูง Zone A
ขัดหลัก ALARA โดยตรง — มีต้นทุนทางจริยธรรมที่ตามมาถึงฉากจบ

nuclearKnowledge: ALARA ไม่ใช่กฎที่บอกให้หลีกรังสีทุกอย่าง
แต่บอกให้สมดุลระหว่างประโยชน์ที่ได้รับกับความเสี่ยง
(ทำ CT scan เพื่อวินิจฉัยโรค = คุ้มค่า แม้ได้รับรังสี 10 mSv)
```

### ENV-03 · เขตรังสีและการจำแนกพื้นที่
```
id: env_radiation_zones
title: เขตรังสี — วิธีจำแนกพื้นที่ปนเปื้อน
unlockEvent: ON_ZONE_A_OPENED
branch: Environment

content:
หลังอุบัติเหตุนิวเคลียร์ พื้นที่จำแนกตามระดับรังสีและความปลอดภัย:

Zone ระบบ IAEA:
• เขตห้ามเข้า (Exclusion Zone): รังสีสูงเกิน 1,000 mSv/ปี — ห้ามเด็ดขาด
• เขตควบคุม (Controlled Zone): 20–1,000 mSv/ปี — เข้าได้ด้วยอุปกรณ์
• เขตปนเปื้อน (Contaminated Zone): 1–20 mSv/ปี — ต้องตรวจสอบ
• เขตปกติ: < 1 mSv/ปี จากแหล่งสังเคราะห์

Zone A ใน Veltara = เขตควบคุม (แร่หายาก + รังสีสูง)
Zone B = เขตห้ามเข้า แต่มีเชื้อเพลิงไตรเทียม — ต้องใช้ Rad-Gear

วิธีปกป้อง:
• Rad-Gear ลดการรับรังสีจากภายนอก (External radiation)
• ไม่กิน/ดม/สัมผัสสิ่งปนเปื้อน (Internal contamination อันตรายกว่า)

nuclearKnowledge: Chernobyl Exclusion Zone (30 กม. รอบโรงไฟฟ้า)
ยังมีพื้นที่บางส่วนที่รังสีสูงเกิน 1,000 mSv/ปี แม้ผ่านมา 40 ปี
เพราะ Cesium-137 มีครึ่งชีวิต 30 ปี
```

### ENV-04 · ผลกระทบรังสีระยะยาว
```
id: env_long_term_effects
title: ผลกระทบของรังสีในระยะยาว
unlockEvent: ON_PHASE_3_STARTED
branch: Environment

content:
ผลระยะยาวของการรับรังสีขึ้นอยู่กับ: ปริมาณ ระยะเวลา ชนิดรังสี และ
อวัยวะที่ได้รับ

ผลทางชีววิทยา:
• Stochastic effects (สุ่ม): มะเร็ง พันธุกรรม — โอกาสเพิ่มแต่ไม่แน่นอน
  ไม่มีขีดจำกัด (แม้ขนาดน้อยก็เพิ่มความเสี่ยงเล็กน้อย)
• Deterministic effects (แน่นอน): ต้องรับรังสีเกิน threshold
  เช่น ต้องรับ > 500 mSv ถึงจะเกิด Acute Radiation Syndrome

อวัยวะที่ไวรังสีที่สุด:
1. ไขกระดูก (สร้างเม็ดเลือด — LD50 ที่ ~4 Gy)
2. ลำไส้เล็ก (เซลล์แบ่งตัวเร็วมาก)
3. ต่อมไทรอยด์ (ดูดซึม Iodine รวมถึง I-131)
4. ดวงตา (ต้อกระจกจากรังสีแกมมา)

บทเรียนจาก Hiroshima/Nagasaki:
ผู้รอดชีวิตที่รับรังสีสูงมีอัตราการเกิดมะเร็งเม็ดเลือดขาวเพิ่มขึ้น
ใน 5–10 ปีแรก แต่ผลต่อทายาทรุ่น 2 ไม่พบอย่างมีนัยสำคัญทางสถิติ

nuclearKnowledge: Linear No-Threshold (LNT) model สมมติว่า
ไม่มี "ปริมาณปลอดภัย" สำหรับรังสีไอออไนซ์ — ทุก mSv เพิ่มความเสี่ยง
เล็กน้อย (แต่นักวิทยาศาสตร์บางส่วนยังถกเถียงกันอยู่)
```
