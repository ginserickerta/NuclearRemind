### 2. ระบบหล่อเลี้ยงพลังงาน (Power Grid System) ✅

ระบบแบบ Frostpunk — Core Tower / Power Plant ปล่อยพลังงานในรัศมี Chebyshev
อาคารนอกรัศมีไม่ produce resource และ dim ลง

#### ไฟล์ใหม่
| ไฟล์ | หน้าที่ |
|---|---|
| `Assets/Scripts/Systems/PowerGridManager.cs` | Singleton คำนวณ BFS coverage |
| `Assets/Scripts/UI/PowerGridVisual.cs` | Overlay สีบน grid (กด P toggle) |
| `Assets/Editor/PowerGridSetup.cs` | Menu: `NuclearReMind/Setup Power Grid System` |
| `Assets/ScriptableObjects/Buildings/PowerConduit.asset` | อาคารใหม่ — relay ขยาย chain |

#### ไฟล์แก้ไข
| ไฟล์ | สิ่งที่เปลี่ยน |
|---|---|
| `BuildingData.cs` | เพิ่ม `powerRange` (int), `isPowerRelay` (bool) |
| `GridManager.cs` | เพิ่ม `PowerConduit` ใน BuildingType enum |
| `EventManager.cs` | เพิ่ม `OnPowerGridChanged`, `RaisePowerGridChanged` |
| `ResourceManager.cs` | skip unpowered buildings ใน `Tick()` |
| `BuildingVisualSpawner.cs` | dim/restore SpriteRenderer color ตาม power state |

#### Logic สำคัญ
- **Primary sources**: PowerPlant (range 2), CoreTower (range 2/4/6/8 ตาม phase)
- **Power Conduit**: `isPowerRelay = true`, range 3 — ต้องอยู่ในระยะ source ก่อนถึง relay ต่อ
- **BFS algorithm**: queue primary sources → mark cells → activate relays ที่อยู่ใน poweredCells → repeat จนไม่มี source ใหม่
- **อาคารที่ไม่มีไฟ**: ไม่ produce food/water/energy/etc. + SpriteRenderer หรี่เป็นสีเทา
- กด **P** เพื่อ toggle overlay แสดง coverage (cyan = powered, red = occupied แต่ไม่ powered)

#### วิธีใช้ใน Unity Editor
1. เปิด Gamescene
2. Menu: `NuclearReMind → Setup Power Grid System`
3. Save Scene (Ctrl+S)
4. Play → วาง PowerPlant ก่อน แล้ว วางอาคารอื่นในรัศมี → กด P ดู coverage

## Power Range ต่อ Building

| อาคาร | powerRange | isPowerRelay | หมายเหตุ |
|---|---|---|---|
| PowerPlant | 2 | false | Primary source |
| CoreTower | 2 (phase 0) → 8 (phase 3) | false | Scaled by PowerGridManager |
| Power Conduit | 3 | true | Relay — ต้องอยู่ในระยะ source |
| อาคารอื่นๆ | 0 | false | ผู้บริโภค |
